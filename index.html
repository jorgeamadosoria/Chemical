Valid input consists of one string of only letters with a max length of 30 characters. The resulting combinations can be real (consisting of existing chemical elements) or fake (with a mix of existing and generated elements). They will be presented 20 at a time below, each batch can change it's color scheme if selection changes.


Valid input consists of a series of 1 or 2 letters, separate by commas. Whitespaces around commas are trimmed, whitespaces between letters invalidate the input. A valid input would be "f,ac, es" without the quotes.

App Info

This application takes words and try to translate them into combination of chemical elements. Not all words can be written with chemical elements: in that case, pseudo-random elements are generated to complete the words.

The elements are printed in one of three selectable scheme colors, 20 at a time.

Aditionally, a particular combination of a word can be printed with chemical elements. This is useful if there is a combination sought, rather than checking all possible ones.

Color schemes
  • Ptable: A far superior visualizer of the periodic table. It contains a lot of information on chemical elements, details on the table and a convenient color scheme that I copied shamelessly.
  • Jmol: This is a visualizer for molecules and complex chemical compounds. It has a convenient color code for most chemical elements except the latest ones.
  • Greyscale: This is just a progressively black to white scheme that gets a lighter background as the atomic weight goes up. The text is dark on the light backgrounds and light on the dark backgrounds.
